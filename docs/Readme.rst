REX Prefix
----------
- 1 per instruction
- usage:
    - specify GPRs and SSE registers
    - specify 64-bit operand size
    - specify extended control registers
- flags:
    - rex.b = 1
    - rex.x = 2
    - rex.r = 4
    - rex.w = 8
    - combined with binary or
- bytecode:
    - 0x40 + flags
    - binary: 0100wrxb
